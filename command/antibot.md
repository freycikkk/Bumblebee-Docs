# Anti-Bot

Enhance your server's anti-bot protection with Anti-Bot.

## Usage

### Manage Anti-Bot Channels

```
?antibot channel <add/remove> <channel mention/id>
```

### Display Anti-Bot Channels List

```
?antibot config
```

### Reset Anti-Bot Channels List

```
?antibot reset
```
